# Login & Register Glassmorphism UI

Bu proje **codebysell** tarafından üretilmiştir.

## Proje Hakkında
Modern, koyu temalı ve cam efektli (Glassmorphism) bir Giriş/Kayıt sayfası tasarımıdır. Arka planda yüksek kaliteli bir video oynatılmakta ve kullanıcı etkileşimlerinde (tıklama, odaklanma) özel animasyonlar bulunmaktadır.

## Özellikler
- **Glassmorphism Tasarım**: Buzlu cam efekti ile modern arayüz.
- **Video Arka Plan**: Tam ekran, yüksek çözünürlüklü video arka planı.
- **Sihirli Değnek İmleci**: Özel mouse imleci tasarımı.
- **Yıldız Efektleri**: Input alanlarına tıklandığında çıkan yıldız animasyonları.
- **Responsive Yapı**: Mobil ve masaüstü uyumlu.

## Kurulum
Bu dosyaları bir web sunucusunda veya doğrudan tarayıcınızda `index.html` dosyasını açarak görüntüleyebilirsiniz.

## About the Project
This is a modern, dark-themed Login/Register page design featuring a Glassmorphism effect. It includes a high-quality full-screen video background and special animations (star effects) on user interactions like clicking or focusing on inputs.

## Features
- **Glassmorphism Design**: Modern interface with frosted glass effect.
- **Video Background**: Full-screen, high-resolution video background.
- **Magic Wand Cursor**: Custom mouse cursor design.
- **Star Effects**: Star animations appear when clicking on input fields.
- **Responsive Structure**: Compatible with mobile and desktop devices.

## Installation
You can view this project by opening the `index.html` file directly in your browser or hosting the files on a web server.
